# Instrucciones:

Solicita al usuario cuántas calificaciones va a capturar.
- Por ejemplo: “¿Cuántas calificaciones deseas registrar?”
- Usa ese número para crear dos arreglos: uno para los nombres y otro para las calificaciones. 
## Captura los datos del alumno.
En un ciclo, pide al usuario:
- El nombre del alumno
- La calificación correspondiente
- Guarda cada nombre en el arreglo de nombres y cada calificación en el arreglo de calificaciones.
- Ambos arreglos deben mantener la misma posición entre nombre y calificación.
Ejemplo: `nombres[0]` corresponde a `calificaciones[0]`.
## Ordena los datos usando Selection Sort.
> [!WARNING]
> Cuando intercambies dos calificaciones, también debes intercambiar los nombres en el arreglo de nombres, para mantener la relación correcta. 
> Muestra los resultados ordenados en consola.
- Imprime primero la calificación y después el nombre correspondiente.
- Verifica que cada alumno siga emparejado con su calificación.
